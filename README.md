# Apart Elementor Video Player

یک ویجت Elementor برای نمایش ویدیوهای آپارات و Self-Hosted

## ویژگی‌ها

- ✅ پشتیبانی از ویدیوهای آپارات (Aparat.com)
- ✅ پشتیبانی از ویدیوهای Self-Hosted
- ✅ تنظیمات کامل ویدیو (Autoplay، Loop، Controls و...)
- ✅ پشتیبانی از Image Overlay
- ✅ پشتیبانی از Lightbox
- ✅ کاملاً Responsive

## نصب

### روش 1: نصب مستقیم از ZIP

1. فایل `apart-elementor-video-player.zip` را دانلود کنید
2. به پنل مدیریت وردپرس بروید
3. به بخش **افزونه‌ها > افزودن** بروید
4. روی دکمه **بارگذاری افزونه** کلیک کنید
5. فایل ZIP را انتخاب و بارگذاری کنید
6. روی دکمه **فعال‌سازی** کلیک کنید

### روش 2: نصب دستی

1. فایل‌های پلاگین را در پوشه `wp-content/plugins/apart-elementor-video-player/` قرار دهید
2. از پنل مدیریت وردپرس پلاگین را فعال کنید

## استفاده

1. پس از فعال‌سازی پلاگین، به ویرایشگر Elementor بروید
2. ویجت **Apart Video Player** را در لیست ویجت‌ها پیدا کنید
3. ویجت را به صفحه اضافه کنید
4. منبع ویدیو را انتخاب کنید:
   - **Aparat**: لینک ویدیو آپارات را وارد کنید (مثال: `https://www.aparat.com/v/ror49jn`)
   - **Self Hosted**: فایل ویدیو را آپلود کنید یا URL خارجی وارد کنید

## نیازمندی‌ها

- WordPress 5.0 یا بالاتر
- Elementor 3.0.0 یا بالاتر
- PHP 7.0 یا بالاتر
